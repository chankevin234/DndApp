# DndApp

# Purpose
This web app will allow DnD players to track their character attributes such as:
1. Stats (str, dex, con, etc.)
1. Items
1. Gold
1. Health

# Structure

## API
https://api.open5e.com/


## Front-End
React Frontend w/ Javascript

## Back-End
Python to call API and parse to MS Excel
C++ to parse Excel
